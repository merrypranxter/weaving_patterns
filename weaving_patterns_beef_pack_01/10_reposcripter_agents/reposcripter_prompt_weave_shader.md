# RepoScripter Prompt: Weave Shader

Use the attached weaving-pattern repository as context. Create a single-file HTML/p5.js or GLSL shader demo that renders a woven textile pattern as visible warp/weft interlacement.

Required features:

- binary drawdown matrix logic
- selectable weave family: plain, twill, herringbone, satin-like, tartan
- warp and weft color arrays
- thread thickness controls
- float shadow / occlusion
- fiber fuzz / yarn noise
- tension wobble
- keyboard or mouse controls for reseed and parameter changes

The result should look structurally woven, not like a flat printed plaid.
