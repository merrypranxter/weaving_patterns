# Repeat Units And Tiles

A weave repeat is the smallest interlacement block that can tile forever without changing the structure.

For shaders, treat the repeat as a texture tile. For prompt systems, describe the repeat as the pattern's grammar.

## Repeat questions

- How many warp ends wide?
- How many weft picks tall?
- Does it tile straight, mirrored, offset, or reversed?
- Are floats aligned, staggered, or interrupted?
- Does color repeat match structure repeat or fight it?
