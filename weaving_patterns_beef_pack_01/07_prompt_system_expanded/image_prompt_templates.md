# Image Prompt Templates

## Short formula

`[subject/use] + [weave family] + [fiber] + [thread color order] + [visible interlacement] + [surface behavior] + [lighting]`

## Example

Extreme macro of a handwoven twill textile in acid-bright yarns, visible warp and weft interlacement, diagonal float ribs, tiny shadows under long floats, fiber fuzz, tension wobble, raking light, procedural textile specimen.

## Shader formula

`Make a single-file shader/p5 sketch that renders [weave family] as a binary drawdown matrix with warp/weft colors, thread thickness, float shadows, fiber fuzz, and interactive knobs for sett, repeat size, tension variance, and palette.`

## Anti-drift line

This must look structurally woven, not printed on fabric. The pattern must come from interlaced threads.
