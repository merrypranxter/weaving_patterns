# Expansion Pack 01 File Map

This pack adds a heavier system around the existing weaving repo.

## New major sections

- `00_foundation_expanded/` — loom logic, draft logic, binary textile thinking
- `01_draft_systems/` — threading, tie-up, treadling, liftplans, drawdowns
- `02_weave_families_expanded/` — specific weave structures and behavior
- `03_color_and_sett_systems/` — thread color ordering, plaid, tartan, checks, stripes
- `04_material_and_surface_logic/` — fiber behavior, floats, sheen, tension, pile, puckering
- `05_shader_templates/` — GLSL snippets and reusable shader logic
- `06_examples_code/` — runnable HTML/p5 demos and JS tools
- `07_prompt_system_expanded/` — image, shader, textile, and style-mixing prompts
- `08_drift_control_expanded/` — anti-bullshit rules for woven structures
- `09_structured_data_expanded/` — JSON/CSV machine-readable specs
- `10_reposcripter_agents/` — agent prompts for Copilot/RepoScripter
