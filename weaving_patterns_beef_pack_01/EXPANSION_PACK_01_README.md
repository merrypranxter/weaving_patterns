# Weaving Patterns Beef Pack 01

Expansion pack for `merrypranxter/weaving_patterns`.

This pack treats the loom as a binary image machine: warp over / warp under, float / bind, draft / repeat, sett / color order, tension / material, structure / surface.

Drop these folders into the repo root. Nothing here replaces the existing README unless you choose to merge it.

## What this adds

- expanded weave family documentation
- drafting and drawdown logic
- textile math / binary matrix rules
- shader templates and GLSL snippets
- p5.js / HTML demos
- JS draft utilities
- SVG draft renderer
- prompt and RepoScripter systems
- structured JSON and CSV data
- anti-drift and repair guides

## Core motto

Every woven pattern is a program. Every thread is a bit. The cloth is the rendered output.
